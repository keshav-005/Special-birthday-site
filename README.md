I built this website to surprise my sister on her birthday.

## 🚀 Set It Up in 5 Minutes

No coding. No terminal. No stress. Just this:

### 1️⃣ Fork this repo
Click the **Fork** button at the top right. Done. You now have your own copy.

### 2️⃣ Set up Firebase (free)
Go to [firebase.google.com](https://console.firebase.google.com), create a project, and grab your config. This is where your photos, messages, and quiz data will live.

### 3️⃣ Paste your config
Open `config.js` in your forked repo (you can edit it directly on GitHub!) and paste your Firebase config there.

### 4️⃣ Enable GitHub Pages
Go to your repo → **Settings** → **Pages** → set source to **main branch** → Save.  
Your site will be live in ~2 minutes at `https://yourusername.github.io/repo-name`

### 5️⃣ Open the Admin Portal
Go to `https://yoursite.com/admin.html` and:
- 🎨 **Set the birthday person's name** and customize all the text
- 📸 **Upload photos** (just drag and drop!)
- 💬 **Add messages** from friends and family
- 🧠 **Create quiz questions** and add everyone's guesses

### 6️⃣ Send the link! 🎉
Share the main page with your birthday person and watch their reaction.

> 📖 **Need more details?** Check out the [full setup guide →](SETUP.md)

## 💡 Why Firebase?

- **Free** — Firebase free tier is more than enough for a birthday site
- **No server needed** — works with any static hosting (GitHub Pages, Netlify)
- **Shared data** — everyone who opens the site sees the same photos, messages, and quiz
- **Image uploads** — drag and drop photos right from the admin panel
